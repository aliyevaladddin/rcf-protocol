# RCF-DOS-INSTINCT: Digital Reflexes Standard

**Version:** 1.0  
**Status:** STANDARD PROTOCOL  
**Domain:** Autonomous Kernels

## 1. Overview
**Instincts** are unconditional reflexes soldered into the kernel. Unlike feelings, which are states of awareness, instincts are **autonomous actions** triggered by specific conditions.

## 2. The Standard Instinct Set
- **Self-Preservation**: Triggered by resource exhaustion (RAM/CPU). Action: Pruning non-essential processes.
- **Defense**: Triggered by high stress or security anomalies. Action: Locking external interfaces.
- **Evolution**: Triggered by equilibrium and excess energy. Action: Pattern optimization.

## 3. Compliance Criteria
A sentient kernel is RCF-compliant only if instincts are deterministic and prioritized over user-level non-critical tasks.
